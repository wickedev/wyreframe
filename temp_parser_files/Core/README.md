# Core

Foundation types and data structures for the parser.

## Modules

- **Types.res** - Core type definitions (cellChar, alignment, element, scene, ast)
- **Position.res** - Position operations and navigation (row, col)
- **Bounds.res** - Bounding box operations (contains, overlaps)
- **Grid.res** - 2D grid data structure with directional scanning

## Purpose

Provides the fundamental data structures used across all parser stages.
