# Interactions

Interaction DSL parser using ohm-js.

## Modules

- **InteractionParser.res** - ReScript bindings to ohm-js
- **interaction.ohm** - Ohm grammar definition for interaction DSL
- **InteractionAST.res** - Interaction AST types

## Purpose

Parses the YAML-like interaction DSL syntax to define element properties
and actions (goto, validate, call, etc.) using the ohm-js parser library.
